AI Programming Learning Hub

A structured, build-first learning path into AI programming, visual machine learning, and interactive coding systems.

Overview

This is a progressive learning series focused on building AI systems from the ground up. Each lesson demonstrates concepts through working code, visual experiments, and iterative development.

What This Series Is

  • Hands-on AI programming tutorials
  • Interactive browser-based experiments
  • Visual machine learning models (perceptrons, classification, etc.)
  • Real debugging, iteration, and system evolution

Learning Structure

  • Game-Based AI Programming — learn fundamentals through building simple apps
  • Perceptron Lab — visual decision boundaries and classification
  • Pattern Recognition — digits, lines, and input interpretation systems
  • Model Evolution — improving systems with memory, undo, and animation

Episodes & Visual Progression

Each block represents a key concept or implementation milestone in the series.

How to Learn From This

Each lesson builds on the previous one. The best results come from watching, rebuilding, and then modifying the system yourself.

  • Watch the concept
  • Rebuild it independently
  • Modify parameters and observe behavior
